# プライバシーポリシー / Privacy Policy

## 日本語

### データ収集について

**PuzCom Solve Time Stats** は、いかなる個人情報も収集しません。

### データの処理

- すべてのデータ処理はブラウザ内で完結します
- 外部サーバーへのデータ送信は一切行いません
- ユーザーの閲覧履歴やアカウント情報にアクセスしません

### 使用するデータ

この拡張機能は、パズコミュ（puzzle.nikoli.com）のページ上に既に表示されている公開データのみを利用します：

- 解答時間
- ユーザー名（表示されている場合）

これらのデータは統計計算とグラフ表示のためにのみ使用され、保存や送信は行いません。

### 権限について

この拡張機能は、以下の権限のみを使用します：

- `https://puzzle.nikoli.com/*/post/*` へのアクセス権限

これは統計パネルとソート機能を提供するために必要な最小限の権限です。

## English

### Data Collection

**PuzCom Solve Time Stats** does not collect any personal information.

### Data Processing

- All data processing is performed locally within your browser
- No data is transmitted to external servers
- We do not access your browsing history or account information

### Data Usage

This extension only uses publicly visible data already displayed on PuzCom (puzzle.nikoli.com) pages:

- Solve times
- Usernames (when displayed)

This data is used solely for statistical calculations and visualization, and is neither stored nor transmitted.

### Permissions

This extension uses only the following permission:

- Access to `https://puzzle.nikoli.com/*/post/*`

This is the minimum permission required to provide statistics panels and sorting functionality.
